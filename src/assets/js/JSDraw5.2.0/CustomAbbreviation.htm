Build Custom Abbreviations

You can use this tool to build a custom list of your atom abbreviations for JSDraw, TouchMol and MolEngine.
1. Use follow JSDraw.Table to build your abbreviations list (Right-click atom to add attach points 1, 2, 3)
2. Once finished, click Save button to save the list into a SDF file
3. To configure JSDraw, TouchMol and MolEngine to use this custom abbreviations list, 
   have the following in to your configuration file (web.config, scilligence.config, or application.config)

    <add key="jsdraw.abbreviations" value="c:\data\custom_abbreviations.sdf" />
