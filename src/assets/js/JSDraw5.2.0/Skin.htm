JSDraw - A Javascript Chemical Structure Editor/Viewer

Please select skin:

Example code to change skin: 
    JSDraw2.Skin.red(); // or blue/green/yellow()
    JSdraw.init();
To customize skin: 
    // customzie skin
    oln.apply(JSDraw2.Skin.jsdraw = { bkcolor: "", bkimg: "", hovercolor: "" });
    oln.apply(JSDraw2.Skin.jssdf = { bgcolor: "", headerimg: "", headercolor: "", rowcolor: "" });
    oln.apply(this.jssdf, JSDraw2.Skin.jsdraw);
    oln.apply(JSDraw2.Skin.dialog = { bkimg: "", bkcolor: "" });

    // create JSDraw
    JSDraw.init();
